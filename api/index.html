VACUUM Supervisor

VACUUM Supervisor is a Java/Gluon application used to monitor and control vacuum subsystems across multiple stations (tube sectors, towers, cryogenic links, remote pumps, gauges, and safety actions).

Scope

This repository contains:

  • UI layers and views (View*, Layer*, SidePopupView*)
  • station and subsystem controllers (Controller*)
  • data definitions and mappings (DataSet*, DataElement, DataTypes)
  • online command/telemetry integration (ControlCommand, ViewData, NotificationData)

The app can be packaged for desktop and mobile targets via the JavaFXPorts/Gluon toolchain.

Technology Base

  • Gluon Java client libraries (Charm/Glisten/Charm Down):
    • https://gluonhq.com/products/mobile/
    • https://github.com/gluonhq/charm
  • JavaFXPorts Gradle plugin (org.javafxports:jfxmobile-plugin):
    • https://github.com/javafxports/javafxmobile-plugin

Public Sanitization

For public sharing, internal online-provider hostnames are replaced with examples.

Run:

./scripts/sanitize_for_public.sh

Optional:

  • EXAMPLE_HOST=online-data-provider.example ./scripts/sanitize_for_public.sh
  • DRY_RUN=1 ./scripts/sanitize_for_public.sh

Build / Run

Examples:

./gradlew :MainApp:runDesktop
./gradlew :MainApp:android
